# bayes-intercept-toy
Bayesian Tool to intercept missiles 
Bayesian Decision-Making Under Scarcity

This interactive model illustrates a simple but powerful idea:

When defensive resources are limited and classification is uncertain, optimal decisions depend not on certainty, but on expected loss.

Each incoming “track” may be a true threat or a decoy.
The defender:

Updates the probability of threat using Bayes’ rule.

Compares expected damage from inaction to the cost of interception.

Fires only if expected loss justifies the shot.

The twist: inventory is finite.

As interceptors are consumed, the effective cost of firing rises.
At some point, depletion becomes more dangerous than uncertainty.

The simulation shows how:

Low base-rate threats

Noisy evidence

Cheap decoys

Finite defensive inventory

combine to create nonlinear outcomes — including sudden loss of control once resources are exhausted.

This is not a tactical model. It is a stylized demonstration of expected-value decision theory under resource constraints.
